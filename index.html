API Challenge: URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • The URL needs to be in the form: "http://fcc-url-shortener-microservice-tiaan999.c9users.io/url".
  • If I pass an invalid URL that doesn't follow the valid "http://www.example.com" format, the JSON response will contain an error instead.
  • When I visit that shortened URL, it will redirect me to my original link.

For example:

  1. Input the URL: "http://fcc-url-shortener-microservice-tiaan999.c9users.io/http://www.google.com"
  2. Receive a shortened url: "http://fcc-url-shortener-microservice-tiaan999.c9users.io/1"
  3. Following the shortened url, will redirect you to your original url at "http://www.google.com"
